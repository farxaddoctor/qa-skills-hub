# API Test Template

## Scenario

Behavior under test.

## Endpoint / Operation

Method and route or operation name. Do not store environment-specific base URLs here.

## Preconditions

Required state, data, auth, or permissions.

## Request

Headers, path parameters, query parameters, and body shape.

## Expected Response

- Status code.
- Required headers.
- Body schema or fields.
- Error format, if applicable.

## Side Effects

Persistence, events, downstream calls, audit records, or no-op expectations.

## Cleanup

Data cleanup or rollback notes.
