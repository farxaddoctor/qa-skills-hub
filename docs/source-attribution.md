# Source Attribution

Use this document to record any copied, adapted, summarized, or referenced third-party material.

No third-party material has been copied into this repository. The current third-party entries are references and placeholders only.

## Attribution Rules

- Mark external sources as references unless content is actually copied or adapted.
- Do not copy large third-party content blindly.
- Prefer short notes and links over pasted source material.
- Record source name, URL, owner, license, retrieval date, and usage type before committing copied or adapted material.
- Respect licenses and usage restrictions.
- Keep copied or adapted third-party material under `third-party/` when practical.
- Do not mix unattributed external wording into `skills/`, `agents/`, `commands/`, `standards/`, or `templates/`.

## Usage Types

| Usage type | Meaning |
| --- | --- |
| Reference | Source is listed for awareness only; no content copied or adapted. |
| Adapted | Ideas or structure are transformed into original wording. Requires attribution. |
| Copied | Verbatim content is included. Requires attribution and license review. |
| Summarized | Source is summarized in original wording. Requires attribution. |
