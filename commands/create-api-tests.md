# Create API Tests

Use `skills/api-testing/SKILL.md` plus the implementation skill for the consuming project.

## Prompt

Design API test coverage from the provided contract, endpoint behavior, or scenario list. If implementation is requested, use the consuming project's existing framework conventions.

## Expected Output

- API coverage matrix.
- Data setup and cleanup notes.
- Contract, auth, negative, schema, and state checks.
- Implementation guidance when requested.
